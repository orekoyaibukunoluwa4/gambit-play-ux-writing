# Gambit Play — UX Writing Case Study

A speculative UX writing portfolio project demonstrating brand voice definition and microcopy across five core user touchpoints in the sports betting industry.

## About

**Gambit Play** is a fictional, original sports betting brand created to showcase end-to-end UX writing skills — from defining a brand's voice to writing the words users actually read inside the product. Gambit Play does not exist as a real company; it was built specifically for this portfolio.

The brand positions betting as a game of strategy and skill rather than luck, appealing to a premium audience who value precision over hype.

## What's Inside

This case study covers:

- **Brand Voice Definition** — personality, tone dimensions, and writing rules that guide every piece of copy
- **Homepage Hero Copy** — first-impression messaging and primary CTAs
- **Registration & Onboarding Flow** — step-by-step microcopy, helper text, and error states
- **Bonus & Promotions Copy** — offer messaging with full compliance language
- **Deposit & Withdrawal UI Copy** — payment flow copy built around trust and clarity
- **Push Notifications & Alerts** — engagement, transactional, and responsible gambling messaging

Each section includes a before/after rewrite comparison and a writer's note explaining the reasoning behind specific word choices.

## Why This Project

UX writing is about more than individual lines of copy — it's about defining how a brand sounds and making sure that voice stays consistent across every touchpoint a user encounters. This project was built to demonstrate that end-to-end thinking: from strategic brand voice decisions down to a single error message.

As a linguist, I approach this work through the lens of how language actually shapes user behaviour — structure, tone, and word choice all carry weight in how trustworthy, clear, or confusing a product feels.
